# Evolution-of-sex-determination-mechanisms-in-turtles
Scripts for the manuscript Zhu & Cai et al., "Evolution of sex determination mechanisms in turtles."
# Contents

This repository is organized into the following subdirectories:

- **Bulk RNA analysis/**  
  R scripts used to generate the figures presented in the manuscript based on the anylysis of Bulk-RNA seq. 

- **Genome information and sex chromosome evolution of P. sinensis/**  
  R scripts used to generate the figures presented in the manuscript about the genomic characteristics of P. sinensis.

For more detailed information, please contact the authors.

---

# Notes

All analyses were performed on a Linux-based high-performance computing (HPC) cluster.  
Many steps in the workflow are computationally intensive and may require:

- Long runtimes (several hours to days)
- Multiple CPU cores
- Substantial memory resources

For detailed requirements and usage, please consult the Method and materials parts of the manuscript and the manuals of respective software tools used in this project.
